# Project Description

A responsive store with a functioning cart built using Create-React-App.

[Live Demo](https://laykenv.github.io/shoppingcart)

## Getting Started 

### 
     git clone https://github.com/LaykenV/shoppingcart.git
     cd shoppingcart
     npm install
     npm start
